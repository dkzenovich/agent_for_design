# 03_sources_and_scope

Для каждого `IN-NEED-XXX` укажите источники и границы включения.

Шаблон:
- IN-NEED: …
- Источники и типы документов/таблиц:
  - <система/хранилище> — типы документов/таблиц — статус (Primary/Secondary/Excluded) — обоснование
- Условия отбора:
  - типы документов/тэги/категории: …
  - состояние (актуален/устарел/черновик): …
  - даты/версии/релизы: …
- Ограничения (ИБ/правовые/лицензии): …
- Пробелы/допущения: …

Отдельно перечислите явные исключения (excluded) и почему.
