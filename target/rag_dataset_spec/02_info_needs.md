# 02_info_needs

Список информационных потребностей в формате `IN-NEED-XXX`.

Шаблон записи:
- ID: `IN-NEED-XXX`
- Роль/персона: …
- Класс вопросов/кейсов (кратко): …
- Примеры вопросов: …
- Критичность: Must/Should/Nice-to-have
- Требования к точности/свежести: …
- Риски/пробелы: …
- Out-of-scope (если есть): …
