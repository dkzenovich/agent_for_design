# pdrs/

Машинные PDR-записи для запуска промптов 10/20/30/40/50.

Правила:
- Имя файла: `YYYY-MM-DDTHH-MM-SS_<prompt>.md`.
- Содержимое — по шаблону из `prompts/90-rag-prompt-run-pdr.md`.
- Фиксируем: входные файлы/версии, допущения, решения, что добавлено/исключено, какие артефакты обновлены (`target/rag_dataset_spec/...`).
- Если изменений нет — пишем об этом явно.
